Promises

What are promises?


            A promise is an object. At the most basic level:

            const myPromise = new Promise((resolve, reject) => {
                // condition
            });
          
*The code above is styled with PrismJS using their Normalize Whitespace plugin.

So, how do we work with promises?


            const myPromise = new Promise((resolve, reject) => {
                let yourNumber = document.getElementById("yourNumber").innerHTML;
                if (yourNumber === "42") {
                  resolve(yourNumber);
                } else {
                  reject(yourNumber);
                }
              });
            
              myPromise
                .then((value) => {
                  alert(`The universe is yours! Your number is ${value}.`);
                })
                .catch((value) => {
                  alert(`Not yet the universe is yours you see. Your number is ${value}.`);
                });
        
            To demo the functionality, change your number using the form below:
0


            Let's understand the code:

                1.) We construct a Promise object arbitrarily named "myPromise" and simultaneously call a function with two arguments–resolve and reject.
                2.) The function gets the value of your number.
                3.) The function checks if your number === "42". If yes, the promise resolves. If no, the promise rejects.
                4a.) If the promise resolves, then only all .then() methods from the myPromise chain will be called, sending us an alert in the browser.
                4b.) If the promise rejects, then only the .catch() method from the myPromise chain will be called, sending us an alert in the browser.
                
                And now we can see the value of promises.

                With promises, we can handle errors and chain callback functions together that execute depending on whether the previous callback functions executed successfully or not.
